    管道分组排序分页¶

    db.example.aggregate([
    {
        $match: {"use_time": {$gte: "2023-02-01 00:00:00", $lte: "2023-02-02 23:59:59"}}
    },
    {
        $group: {_id: "$col", "count": {$sum: 1}}
    },
    {
        $sort: {"col": -1}
    },
    {
        $skip: 1
    },
    {
        $limit: 10
    }
])

match: 查询条件.
group：分组
    _id: 分组条件
    count: 用于统计组内数据量
sort：排序
skip：跳过多少条
limit：限制条数
